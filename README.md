# HyperDPIP
此為一個非官方的DPIP分支，整合了 HyperOS 3 的 HyperIsland 與 焦點通知。

## 使用前須知
- 僅限搭載 HyperOS 的裝置
- 請確保手機已root並且安裝 LSPosed
- 安裝 HyperCeiler

HyperCeiler -> System UI -> 通知與控制中心 -> 移除焦點通知白名單 [OFF -> ON]
